# Optimizing-Marketing-Campaigns-Using-Predictive-Analytics
This repository contains the complete implementation of the project focused on predicting customer subscription using machine learning models and improving marketing campaign targeting. The project uses the UCI Bank Marketing dataset and applies Logistic Regression, Random Forest, and XGBoost.

# Bank Marketing Predictive Analytics

## Models
- Logistic Regression
- Random Forest
- XGBoost

## Features
- SMOTE for imbalance
- Model evaluation with multiple metrics
- Visualization of results
- Top-K targeting analysis

## Run
pip install -r requirements.txt
python main.py
